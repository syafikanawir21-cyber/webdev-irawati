Portfolio Website irawati

Portfolio ini adalah proyek website sederhana yang dibuat menggunakan HTML, CSS, dan JavaScript. Website ini dirancang sebagai media untuk menampilkan informasi pribadi, keahlian, pengalaman, serta proyek-proyek yang pernah dibuat.
Fitur Utama

Halaman Responsive, Tampil dengan baik di berbagai ukuran layar.

Navigasi Interaktif, Menggunakan JavaScript untuk membuat navigasi lebih dinamis.

Animasi & Style Modern, Menggunakan CSS untuk tampilan yang menarik dan profesional.

Struktur Rapi, Dibangun dengan HTML yang terorganisir.
